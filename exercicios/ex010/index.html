Links locais,externos e download de arquivos

Os links ou hyperlinks são ligações que esistrm entre paginas. Vamos ver alguns exemplos do uso das ãncoras.

Os links locais apontam para documentos que estão no seu próprio site. Acesse outra página nesse servidor clicando no link anterior

Também podemos colocar documentos em outras pastas locais. Acesse uma página em outra pasta clicando no link anterior.

Também é possível acessaar outros sites usando âncoras. Por exemplo,vôce pode criar um link para acessar o youTube (não se esqueça de clicar no botão "VOLTAR" do seu navegador depois,para voltar para cá)

O link acima faz o usuário ir pra outro site e abandona o meu. O ideal é que ele abra outros sites em uma nova aba. Acesse o Youtube agora em uma nova aba em branco.

Os links também podem ser usados para fazer o download de arquivos de outros formatos como zip,pdf,rar,etc.Por exemplo,abra aqui um livro em PDF de exemplo ou baixe a versão compactada do livro.

OBS:Clique sempre em exibir o código fonte para ver como tudo está sendo criado
